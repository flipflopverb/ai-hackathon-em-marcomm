# AI Hackathon EM MarComm

## Purpose

Build a branded University of Utah discovery game for prospective students. Players take a short guided tour as `Swoop` across a stylized topographical campus-and-city map, answer questions about student life and academics, receive a `profile` and `academic_interest`, and then continue to an embedded Slate RFI with non-PII game metadata passed through hidden fields.
