# AI Hackathon EM MarComm

## Purpose

Build a branded University of Utah discovery game for prospective students. Players take a short tour across a stylized topographical campus-and-city map, answer questions about student life and academics. durring the tour they grow a `Swoop` (university of utah mascot) tamogachi from egg to adult and collect items at each stop to help grow it, receive an `academic_interest`, and then continue to an embedded Slate RFI with non-PII game metadata passed through hidden fields.
