## Core Type System

### 1) Primary Display Typeface

- Typeface: **Rigid Square**
- Intended role: display headlines and high-impact campaign moments.
- Required treatment: set in **all caps**.
- Character: hard bevels and angular forms that align with Block U / mountainous geometry.

### 2) Body Copy Typeface

- Typeface: **Myriad Pro**
- Intended role: body copy and most functional reading text.
- Character: legible, flexible, and suitable across long and short copy blocks.

### 3) Supporting Serif Accent

- Typeface: **Mokoko**
- Intended role: selective supportive headline/brand voice moments.
- Character: slab serif with harder edges plus a more welcoming tone.

## Fallback Rules

- If **Rigid Square** is unavailable:
  - Canva substitute: `Rubik One`
  - Web substitute: `Vitesse`
  - Other applications: `Arial`
- If **Myriad Pro** is unavailable:
  - Preferred substitute: `Source Sans Pro`
  - Acceptable substitute when needed: `Arial`
- If **Mokoko** is unavailable:
  - Preferred substitutes: `Sanchez`, `Georgia`

## Practical Usage Guidance

- Keep display typography bold and concise; avoid long headline blocks in display faces.
- Introduce variety with weight, size, and tracking to avoid static type treatments.
- Balance expressive display type with restrained body typography for readability.
- For button text, choose campaign fonts that maximize CTA contrast against nearby headline treatments.
- Preserve accessibility and legibility across responsive breakpoints and smaller viewports.
