# Uofu Enrollment Branding

## Overview

Apply the University of Utah enrollment campaign strategy and visual system to real deliverables. Produce campaign-ready messaging and design guidance that stays consistent with audience priorities, tone, and brand rules.

## Quick Start

1. Read [references/strategy-guidelines.md](references/strategy-guidelines.md) before drafting any output.
2. Read [references/color-palette.md](references/color-palette.md) for exact color values when producing or reviewing visual specs.
3. Read [references/typography-guidelines.md](references/typography-guidelines.md) for exact type system and fallback rules.
4. Read [references/asset-library.md](references/asset-library.md) when selecting source files for production-ready layouts.
5. Identify audience, funnel stage, and the single primary message.
6. Build copy and layout guidance using the voice, positioning, and visual rules in the reference.
7. Run the QA checklist in the reference before finalizing.

## Workflow

### 1) Define the assignment

- Ask for: audience, channel, objective, CTA, constraints (length/size), and required assets.
- If details are missing, make the safest assumption and label assumptions explicitly.
- Keep one central message per deliverable unless the user explicitly requests multi-message content.

### 2) Build the message structure

- Ground the work in the campaign core value proposition and pillar logic in the reference.
- Use specific proof points, not vague claims.
- Keep language accessible and natural; avoid internal jargon and acronym-heavy copy.

### 3) Shape campaign voice and headlines

- Write with upward momentum, support, and possibility while staying clear and concrete.
- Prefer concise, bold headlines that can carry emotional tone quickly.
- Use "climb/rise/next peak" framing carefully and naturally; avoid over-technical climbing terms.

### 4) Apply design system decisions

- Use primary color dominance and secondary colors as accents.
- Prefer sharp-corner button and container treatments; avoid rounded-first styling.
- Use 45-degree geometric cues, textures, and photography perspective rules where appropriate.
- Preserve readability and accessibility, including contrast checks.

### 5) Deliver outputs by channel

- For copy tasks: provide headline options, body copy, CTA options, and rationale tied to audience.
- For design direction: provide layout priorities, hierarchy, typography usage, color usage, and asset guidance.
- For campaign QA: report non-compliance and provide precise remediation edits.
